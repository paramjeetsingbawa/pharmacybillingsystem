This tool is designed for Product monitoring and sales monitoring purpose. 
                                             
Following are the Details on pages and there working

Invoice And Product Billing : (For Billing and Invoice Generation)
                                             
    > This contains product search, based on product name, product id, formula, batch no. Search will be partial search
    > From here user can select a product and add the purchasing quantity and generator sales invoice
    > All sales invoices can be found at sales history section                                               
    > User can also search the products, Expiring for this month 
    > Generate Invoices with complete product list
    > GST support

Purchase Order : (For ordering products to vendor)
    > Search for products
    > Add multiple products in one order
    > Generate Purchase order receipt
    > New purchase order will automatically increases the Quantity / Inventory or your products
                                 
SALES HISTORY :

    > User can track sales entry, Product added in sales, who added those etc..
    > User can delete sales data from this screen
    > User can generate the new invoices from old sales data also

INVENTORY VIEW :

    > User can track Product which has expired last month 
    > User can track Product which will expire this month 
    > User can track Product which is having quantity less than Configured number ( for example #2 ) 
                                                
SALES CHART : 
                                              
    > A graphical representation on day wise sales performed. 

SALES SUMMARY : 

    > Per day basis tabular format data of sales in avon.
